# TypedArrays

Generics replacement for PHP. 
Implement an array of a defined type.

## Requirements

Requires PHP >= 7.4

## Installation

```
composer require chemaclass/typed-arrays
```

### Git Hooks

Enable the git hooks with `./tools/git-hooks/init.sh`
